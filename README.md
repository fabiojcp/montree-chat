# Mini Chat

Aplicação de mini-chat construída com React + Vite e Styled Components.

## Funcionalidades

- Exibição de mensagens iniciais
- Envio de novas mensagens
- Simulação de recebimento de mensagens em tempo real (a cada 5 segundos)
- Indicador de "está digitando..."
- Rolagem automática para a mensagem mais recente
- Definição de nome de usuário (salvo no localStorage)
- Interface visualmente distinta para mensagens enviadas e recebidas

## Instalação e Execução

```bash
npm install
npm run dev
```
